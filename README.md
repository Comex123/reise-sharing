# MatchTrip

Kleine statische Demo-App, in der Menschen passende Reisen finden, solo planen oder gemeinsam verreisen koennen.

## Lokal starten

Die Website benoetigt kein Build-Tool. Einfach `index.html` im Browser oeffnen.

## Struktur

- `index.html` Startseite
- `reisen.html` Reiseuebersicht
- `reise-detail.html` Detailansicht einer Reise
- `reise-erstellen.html` Formular fuer neue Reisen
- `profil.html` Demo-Profil mit Reisen und Anfragen
- `css/` Styles
- `js/` Demo-Daten, Storage und Interaktion

## Deployment

Fuer GitHub Pages ist bereits ein Workflow unter `.github/workflows/pages.yml` vorbereitet.
